# Shirin Magazin - Backend

## Installation

1. Create virtual environment:
```bash
python -m venv venv
source venv/bin/activate  # macOS/Linux
```

2. Install dependencies:
```bash
pip install -r requirements.txt
```

3. Create .env file:
```bash
cp .env.example .env
```

4. Run the server:
```bash
python main.py
```

Server will run on http://localhost:8000
